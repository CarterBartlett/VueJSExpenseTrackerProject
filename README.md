# vue-expense-tracker

Vue expense tracker based on and expanded from [Brad Traversy's Vue3/Composition API Project](https://youtu.be/hNPwdOZ3qFU).

## Project Setup

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Compile and Minify for Production

```sh
npm run build
```
